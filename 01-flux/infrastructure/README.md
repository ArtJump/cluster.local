# 🏗️ Базовая Инфраструктура (Infrastructure)

В этой директории описываются базовые и системные компоненты кластера, на котором будут работать пользовательские приложения (базы данных, веб-серверы и т.д.).

Сюда устанавливаются контроллеры, операторы и провижинеры, которые расширяют стандартные возможности чистого Kubernetes.

## 📂 Структура

Используем подход **Kustomize**. В корне этой папки лежит `kustomization.yaml`, который указывает Flux, какие именно поддиректории нужно применить к кластеру.

Текущие компоненты:
* 💾 **[storage](./storage/)** — Управление локальными дисками (Нарезка Persistent Volumes на базе SSD и HDD).
